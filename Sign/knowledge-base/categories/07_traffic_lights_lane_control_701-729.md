# Part 7: Traffic Lights & Lane Control Signs (701–729)

**Hebrew:** תמרורי רמזורים ובקרת נתיבים
**Sign range:** 701–729 (including illuminated variants פ723–פ726)
**Shape/Color:** Signal lamp housing — Red / Yellow / Green / White
**General rule:** Traffic lights override other traffic signs at intersections. Signals displayed above a specific lane apply only to that lane.

---

## Vehicle Traffic Lights (701–706)

### Sign 701 — RED LIGHT (STOP)

| Field | Value |
|---|---|
| **Number** | 701 |
| **Hebrew Term** | אור אדום — עצור |
| **Category** | Traffic Lights & Lane Control |
| **Shape / Color** | Circular red signal |
| **Image** | `../../images/sign_701.png` |
| **Meaning** | Red signal: stop. Do not enter the intersection or cross the stop line. |
| **Applicability** | Applies to all vehicles facing the signal. If located above a specific lane, applies to that lane only. |
| **Variants** | Flashing red = same as STOP sign (full stop required before proceeding when safe). |
| **Cross-references** | 702, 703, 704, 705, 706, 302 |
| **Paired with** | 702 (yellow), 703 (green) |
| **Notes** | Red + Yellow simultaneously = signal about to change to green; do not proceed yet. |
